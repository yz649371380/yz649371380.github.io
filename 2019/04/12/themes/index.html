Hexo+github快速搭建个人博客

四月 12, 2019
  • Hexo

  昨天用hexo+github搭建了我的第一个个人博客，给大家分享一下过程，若有什么错误的地方，欢迎指出！

  先看看首页图，选择的是我比较喜欢的一个主题，主题的配置将在下面说明！

1.首先，你需要在github上建一个自己的仓库

  git相信大家都不陌生，注册什么的我就不多说了。登录之后，点击右上角的加号，选择New repository，进入下一步。

注意： 库的名字要和你的用户名一致

  创建成功后，默认的域名就是http://yourusername.github.io，如果你想要一个个性的域名，可以购买一个，然后指向你的git仓库。git仓库搭建完后，接下来搭建我们的开发环境。

2.开发环境搭建

  搭建Hexo开发环境需要用到node.js和git，由于时间问题，node.js和git的安装请自行解决，这里不再多说，若有什么问题，请在下面留言。

  安装完之后，需要配置我们的git仓库，授权ssh。右键桌面，打开git bash，输入ssh-keygen -t rsa,按回车三下

  这个操作将在我们的C盘用户目录下的.ssh目录生成id_rsa和id_rsa.pub这两个文件，前者是密钥，后者是公钥，用记事本打开id_rsa.pub，复制其中的全部内容，添加到GitHub上，进行授权。

  点击New SSH key，将你复制的的内容填入里面保存，然后在git bash中输入ssh -T git@github.com进行测试，返回Hi username ！You’ve successfully ……说明你已经成功啦！

3.安装Hexo

  在任意一个地方新建一个文件夹，在文件夹中右键选择git bash，输入npm install -g hexo安装hexo即可。安装过程中可能出现WARN警告，可以不用管，安装完之后输入hexo init <项目名称>初始化，cd <项目名称>，npm install，最后输入hexo s。在浏览器中输入 http://localhost:4000 ，就可以打开我们的博客啦！

$ hexo init <folder>
$ cd <folder>
$ npm install
$ hexo s

  修改配置文件，打开根目录下的_config.yml，在deploy栏添加你的git地址：

  配置完之后就可以上传我们的代码到git上面，在命令窗中输入npm install hexo-deployer-git –save,再输入hexo g，然后再hexo d，就可以将我们public里面的代码上传到我们的git仓库，然后通过之前配置的域名就可以打开我们的博客啦！！！

$ npm install hexo-deployer-git --save
$ hexo g
$ hexo d
